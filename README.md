# 清欢AI聊天插件

一款使用AI编辑器开发的Obsidian AI聊天插件，提供智能对话和内容创作功能。

## ✨ 主要功能

- 🤖 **智能对话** - 支持多种AI模型，流畅的聊天体验
- 📝 **多模型支持** - 兼容OpenAI、Gemini、DeepSeek等主流AI模型
- 🎛️ **灵活配置** - 自定义API密钥、代理设置和系统消息
- 📚 **历史记录** - 自动保存对话历史，支持星标和分类管理
- 🎨 **拖拽调节** - 输入框高度可拖拽调整，记忆用户偏好
- 🔒 **隐私保护** - 本地存储，数据安全可控

## 🚀 快速开始

### 安装

1. 下载插件文件到Obsidian插件目录
2. 在Obsidian设置中启用"清欢AI聊天"插件
3. 配置API密钥和模型设置

### 基本使用

1. **配置API**：在插件设置中添加你的API密钥
2. **选择模型**：从下拉菜单选择AI模型
3. **开始对话**：在输入框中输入消息开始聊天

## ⚙️ 配置说明

### API设置
- **API密钥**：支持多个密钥管理
- **基础URL**：自定义API服务地址
- **代理设置**：支持HTTP/SOCKS代理

### 模型配置
- 支持GPT、Claude、Gemini、DeepSeek等
- 可调节温度和最大token数
- 支持流式输出和常规输出

### 系统消息
- 预设多个角色模板
- 支持自定义系统消息
- 可保存和管理多个提示词模板
